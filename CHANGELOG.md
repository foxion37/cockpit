# Changelog

## 0.2.0

- Added the `kr-batch` profile for a concise Korean Cockpit surface with batch detail files under `docs/batches/`.
- Added `cockpit validate --profile kr-batch` for strict document quality checks.
- Added opt-in strict hook validation through `COCKPIT_STRICT=1`.
- Documented the Mermaid `classDef` color policy and the split between default `.cockpit/` output and `kr-batch` output.

Release status: prepared locally; npm publication has not been performed.
